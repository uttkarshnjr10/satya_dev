Featured Project

GuestGuard: Centralized Verification System

GuestGuard is a full-stack web application designed to centralize hotel guest registration and provide law enforcement with secure, region-wise access to verified data. This system addresses the critical need for a unified platform to enhance public safety and streamline investigations.

View Live Project

Key Features:

  • Role-Based Access Control: Separate, secure dashboards for Hotel Staff, Police, Regional Admins, and a Super Admin.
  • Secure Guest Registration: Hotels can register guests with mandatory ID proof and live photo capture.
  • Advanced Police Search: Law enforcement can search for individuals by name, ID, or phone number within their designated jurisdiction.
  • Admin Oversight: Admins can manage user onboarding, monitor system activity, and ensure data integrity.
